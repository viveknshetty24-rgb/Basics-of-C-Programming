Functions Programs

A function is a reusable block of code that performs a specific task. It helps in reducing code repetition, improving readability, and making programs modular.

This folder contains basic function programs in C.

`Programs`

- `square_of_num.c → Takes a number as input and returns its square using a function`
- `cube_of_num.c → Takes a number as input and returns its cube using a function`
