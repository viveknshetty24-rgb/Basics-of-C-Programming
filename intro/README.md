# 💻 C Programming Basics

This repository contains my beginner-level C programs created while learning the fundamentals of C programming.

## 📂 Files Included

- `data_types.c` → Examples of different data types in C  
- `first_program.c` → My first C program  
- `logical_operators.c` → Usage of logical operators (`&&`, `||`, `!`)  
- `student_details.c` → Basic structure-based or input/output program  
- `user_input.c` → Taking input from user using `scanf`
