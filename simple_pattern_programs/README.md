This folder contains simple C programming pattern problems using nested loops, created for practice and improving logic building skills in C.

📌 Pattern Programs:

`- square_pattern.c → Generates a square star pattern using nested loops with input validation for equal dimensions.`

`- left_aligned_triangle.c → Prints a left-aligned right-angled triangle pattern using nested loops based on user input, with each row containing incrementally increasing stars starting from 1 to n.`

`- pyramid.c → Prints a center-aligned pyramid star pattern using nested loops based on user input, where each row contains an increasing odd number of stars centered with leading spaces, forming a symmetrical pyramid shape from 1 to n rows.`
