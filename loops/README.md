# 🔁 Loops in C

This project demonstrates the implementation of different types of loops in C:

* While loop
* Do-While loop
* For loop
* Jump Statements
  ->break
  ->continue

Each loop is explained using simple programs to understand its working and usage.

---

## 🔹 1. While Loop

Used when the condition is checked **before execution** and the number of iterations is not fixed.

**Programs:**

* `numbers.c `-> Prints numbers from 1 to 10.
* `numbers_rev.c `-> Prints numbers from 10 to 1.

---

## 🔹 2. Do-While Loop

Executes the loop body **at least once** before checking the condition.

**Programs:**

* `sum_all.c` -> Returns the sum of all numbers from 1 to the entered target number.
* `sum_of_digits.c`-> Returns the sum of all digits of the entered number.

---

## 🔹 3. For Loop

Used when the number of iterations is **known beforehand**.

**Programs:**

* `even_sum.c` -> Returns all even numbers between 2 and 20 and also returns their sum.
* `odd_sum.c` -> Returns all odd number between 1 and enterd number and also returns their sum.
* `table.c` -> Returns table of the enterd number.

##🔹 4. Jump Statements

Used to control the flow of loops instantly by skipping or terminating iterations.

**Programs:**

* `break_example.c` -> Prints numbers from 0 to 5 but stops the loop when it reaches 3 using break.
* `continue_example.c` -> Prints numbers from 0 to 10 but skips 5 using continue.

---

## 🎯 Objective

* Understand different types of loops
* Learn how to control program flow
* Build basic logic using repetition

---

## 📝 Note

All programs are written in C and focus on clarity and understanding of looping concepts.
